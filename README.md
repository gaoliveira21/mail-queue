# :mailbox: Mail queue

**A simple example of send mails with [bee queue](https://github.com/bee-queue/bee-queue), [nodemailer](https://nodemailer.com/about/) and [handlebars](https://handlebarsjs.com/)**

## Requirements

- NodeJS (12.17.0)
- Docker and Docker Compose
- Yarn (optional)

---

## How to run

- Install dependecies:

```
  yarn // or npm i
```

- Run redis container

```
  docker-compose up -d
```
